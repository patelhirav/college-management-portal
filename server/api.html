College Management System API Documentation

Base URL: http://localhost:5000/api

Authentication Endpoints

POST /auth/login No Auth
Authenticate user and get access token
POST /auth/forgot-password No Auth
Initiate password reset process
POST /auth/verify-otp No Auth
Verify OTP for password reset
POST /auth/reset-password No Auth
Reset user password
POST /auth/student-signup No Auth
Register a new student
GET /auth/departments No Auth
Get list of all departments

Super Admin Endpoints

GET /super-admin/profile JWT Required
Get super admin profile
GET /super-admin/hods JWT Required
Get list of all HODs
POST /super-admin/create-hod JWT Required
Create a new HOD
GET /super-admin/departments JWT Required
Get list of all departments (admin view)

Admin (HOD) Endpoints

GET /admin/profile JWT Required
Get admin profile
POST /admin/bio JWT Required
Add admin bio information
PUT /admin/bio JWT Required
Update admin bio information
GET /admin/department JWT Required
Get department information
POST /admin/create-professor JWT Required
Create a new professor
POST /admin/create-subject JWT Required
Create a new subject
POST /admin/assign-professor JWT Required
Assign professor to subject

Sub-Admin (Professor) Endpoints

GET /sub-admin/profile JWT Required
Get sub-admin profile
POST /sub-admin/bio JWT Required
Add sub-admin bio information
PUT /sub-admin/bio JWT Required
Update sub-admin bio information
GET /sub-admin/subjects JWT Required
Get list of assigned subjects
POST /sub-admin/create-task JWT Required
Create a new task (supports FormData)
PUT /sub-admin/tasks/:taskId JWT Required
Update a task (supports FormData)
GET /sub-admin/tasks JWT Required
Get list of all tasks
GET /sub-admin/task-status/:taskId JWT Required
Get status of a specific task

Student Endpoints

GET /student/profile JWT Required
Get student profile
POST /student/info JWT Required
Add student profile information
PUT /student/profile JWT Required
Update student profile (supports FormData)
PUT /student/profile/photo JWT Required
Update student profile photo (FormData)
GET /student/subjects JWT Required
Get list of student's subjects
GET /student/tasks JWT Required
Get list of student's tasks
PUT /student/task-status/:taskId JWT Required
Update task status (supports FormData)
